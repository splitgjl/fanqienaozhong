# 番茄专注钟

一款零依赖、响应式的网页版番茄钟，帮助你用清晰的工作与休息节奏保持专注。

## 功能

- 专注、短休息、长休息三种计时模式
- 自定义各模式时长与每日目标
- 自动切换休息/专注模式
- 当前任务记录与本地持久化
- 今日番茄数、专注分钟和连续天数统计
- 声音与桌面通知提醒
- 键盘空格键快捷开始或暂停
- 响应式布局和无障碍支持

## 本地运行

这是一个纯静态项目，直接打开 `index.html` 即可使用。也可以使用内置的零依赖预览服务器：

```powershell
npm start
```

然后访问 `http://localhost:8000`。

## GitHub Pages

推送到 GitHub 后，在仓库的 `Settings > Pages` 中选择从分支部署，并将分支设置为 `main`、目录设置为 `/ (root)`。

## 技术栈

HTML、CSS、原生 JavaScript，无构建步骤、无运行时依赖。
